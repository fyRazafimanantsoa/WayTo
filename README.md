# WayTo
This a web app that will allow you to build a roadmap for your work project, learning project, skill learning journey.

***Purpose***:  Help people developing a project in a more visual way and allow them track progress more efficiently with visual map of where they are in the project.

***Scope***: Anyone with a clear project in mind like PM, Programmer, CEO, Auditor, ISO, solopreneur , entrepreneurâ€¦. comprising people with clear ideas but foggy starting point or not sure on further steps of their project.

***Features***: 

_Asking user goal.s and/or ideas for the project. Input for full description

_Generating a visual view of the project progress in the dashboard.

_ Aligned steps of the project align chronologically. Steps are dissected and detailed as possible to allow the user to see what should be done within a specific step

_Explaination section for of why steps are align as established and tips on how it can be performed
